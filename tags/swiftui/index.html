SwiftUI - View Builders

@ViewBuilder is a parameter attribute that constructs child views from closures in a resulable manner. @ViewBuilder is one of the possible result builders provided in Swift language. Every element in SwiftUI, uses @ViewBuilder to build it’s child Views. Like HStack: struct ContentView: View { var body: some View { HStack{ Text("First Text") Text("Second Text") } } } And if we check the defination of HStack, We can see the it gets content using the @ViewBuilder.
